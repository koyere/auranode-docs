# AuraNode Docs

Public documentation for [AuraNode](https://auranode.app), built with
[Docusaurus](https://docusaurus.io). Published at **https://docs.auranode.app**.

## Develop

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

The static site is generated in `build/`.

## Contributing

Docs live in `docs/`. Edit the Markdown, open a pull request. All content is in English.
